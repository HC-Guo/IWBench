The `div` elements with the `class` attribute set to "flowchart" and "node" define the overall layout and individual nodes, respectively. The `data-id` attribute is used to store unique identifiers for each node, which can be used to create connections between them. The `node-content` `div` element contains the content of each node, including the topic and subtopics.
