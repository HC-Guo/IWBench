What is a Class?

There is more than just the element of a class. It is a way you can group elements with the same class name. You can use the class name to affect the paragraph below, text inside the same class, and the CSS for the class.

Layout tip

Use the layout direction property to change the way the text is aligned in the series of blocks. You can change the text alignment by changing the parent element of this text.

Additional tips

Use the flexbox design to create a layout that is more responsive. Use the class of the selection to style the selected text. Use the property of the width to center the text horizontally in the element.
